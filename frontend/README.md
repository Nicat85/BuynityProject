# Buynity Frontend

A minimal Next.js app for the Buynity backend.

## Setup

1. Copy env file:

```bash
cp .env.local.example .env.local
```

2. Install dependencies:

```bash
npm i
```

3. Run dev server:

```bash
npm run dev
```

Open http://localhost:3000.

## Config
- `NEXT_PUBLIC_API_BASE_URL`: Your backend URL (e.g., http://localhost:5000)
